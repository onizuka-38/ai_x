# 예외처리
- 예외가 날 가능성이 있는 부분에 대해 미리 예상하고, 
  그에 대한 처리를 프로그래밍 하는 것(이유 : 좀 더 안정적으로 실행)

try :
    예외가 발생할 수도 있는 문장들
except 예외타입1 :
    해당 예외가 발생할 경우 실행할 문장
except 예외타입2 :
    해당 예외가 발생할 경우 실행할 문장
---------------------------------------------------------------
try:
    예외가 발생할 수도 있는 구문
except 예외타입 [as e]:
    예외가 발생되면 수행할 구문
else:
    try절을 수행하다 예외가 발생 안되면 else 절을 한번 수행
finally:
    예외가 발생 안 하면 try-else-finally 절 수행
    예외가 발생하면 try 수행 중 except - finally절 수행

# 추상클래스
- 추상클래스 : 객체를 생성할 수 없는 클래스. 상속받을 클래스의 포맷 제공
- 추상메소드 : 호출할 수 없는 메소드
