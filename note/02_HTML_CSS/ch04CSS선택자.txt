CSS 선택자는 HTML 요소를 선택해 스타일을 지정하는 기능이다.
주요 종류는 다음과 같다:

1. 태그 선택자: 특정 태그에 적용.
2. 전체 선택자 (*): 모든 요소에 적용.
3. id 선택자 (#): 특정 id에 적용.
4. class 선택자 (.): 특정 class에 적용.
5. 혼합 선택자: 태그 + id/class 조합.
6. 속성 선택자: 특정 속성을 가진 요소에 적용.
7. 후손/자손 선택자: 계층 구조 기반 선택.
8. 동위 선택자 (+, ~): 형제 요소 선택.
9. 반응 선택자 (:hover): 마우스 올렸을 때 스타일.
10. 상태 선택자 (:focus 등): 입력 요소의 상태 기반.
11. 구조 선택자 (:nth-child 등): 문서 구조 기반 선택.
12. 링크 선택자: a[href] 요소에 적용.
13. 부정 선택자 (:not): 특정 조건 제외 요소 선택.
