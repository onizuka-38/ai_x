한국건설산업연구원(건산연)은 15일 발간한 '2008년 금융위기와 비교한 최근 건설경기 진단과 대응 방안' 보고서에서 2008년 금융위기 당시와 현재 건설경기를 비교한 여러 통계를 기반으로 이같이 밝혔다. 건산연에 따르면 건설경기 선행지표인 건설수주(경상)는 2023년 전년 대비 16.6% 감소해 금융위기 당시인 2008년 -6.1%보다 눈에 띄게 감소폭이 컸다. 건축 착공면적 역시 2008년에는 전년 대비 22.2% 감소했으나 2023년에는 -31.7%로 크게 줄었다.
